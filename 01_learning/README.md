# 01 · Learning — 12-Day Modern AI, GenAI & Agentic Systems Programme

The foundation module for this capstone. Each day pairs a **written walkthrough**
(`README.md` in that day's folder) with the **hands-on notebook** you code alongside it —
progressing from classical ML all the way to multi-agent orchestration. The five projects
in this repo (`02_…` through `06_…`) apply everything below.

## How to use

Open a day's `README.md`, read top-to-bottom, and type each cell of that day's notebook
yourself as you go. Most notebooks are designed to run in **Google Colab** (nothing to
install locally); a few use the **Groq** API — get a free key at
<https://console.groq.com/keys>.
